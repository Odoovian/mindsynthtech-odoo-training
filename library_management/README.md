# Library Management Module – Odoo

This project is developed as part of **Mindsynthtech Odoo Training**.
The module demonstrates basic Odoo custom module development including models, views, menus, and additional fields.

---

# Module Overview

The **Library Management Module** allows users to manage books inside Odoo.

# Features:

* Create and manage books
* Add book category (Selection field)
* Add book price (Float field)
* List and Form views
* Custom menu integration
* Fully installable Odoo module

---

# Installation Steps

1. Copy the module folder into your Odoo `custom_addons` directory.
2. Restart Odoo server.
3. Activate Developer Mode.
4. Go to **Apps**.
5. Click **Update Apps List**.
6. Search for **Library Management**.
7. Click **Install**.
